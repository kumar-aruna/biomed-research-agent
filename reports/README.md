# Sample reports

This folder contains example reports generated by the BioEvidence Research Agent. They are checked in so you can see what the agent actually produces without having to run the full pipeline yourself.

Each report was produced by calling `research(question)` in `Research_Agent.ipynb`.

## What's in a report

Every report has the same structure:

1. **Header** — the original query, the date, and total runtime.
2. **Agent trace** — step-by-step log of which tools the planner called in what order.
3. **Disease Overview** — description from OpenTargets.
4. **Top Gene Targets** — ranked table of associated genes with Ensembl IDs and scores.
5. **Literature Findings** — synthesized claims with PMID citations, drawn from BM25-ranked passages.
6. **Additional Insights** — context from web search (clinical trials, recent news).
7. **Summary** — short wrap-up.

## Reproducing a report

```python
from Research_Agent import research   # or run the notebook cell
research("What gene targets are associated with Parkinson's disease?")
```

New reports are saved to this folder with a filename derived from the query and date.
